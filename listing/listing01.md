Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод: [77, 78, 79]

Слас b это срез с слайса a  от первого до четвертого элемента включительно, поэтому значения b берутся от слайса a

```
